Detailed Description

quokka::InstructionInstance

Localization for an instruction instance

An instance of an instruction is associated to a tuple (chunk, block, instruction index). Remember that instructions are deduplicated so no address field is available but we still need to attach some elements to a specific instance of an instruction (e.g. a comment).
