Detailed Description

quokka::Segments

MultiMap-like collection of segments that stores them like {segment_id -> Segment}. Ida being Ida, we cannot trust the sel_t sel to be unique per segment. Despite being written in the documentation that it should be unique, there are plenty of counter-examples, even in simple elf x86 binaries. This is why we have to rely on multimaps instead of a cleaner approach with maps.
