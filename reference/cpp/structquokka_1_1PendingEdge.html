Detailed Description

quokka::PendingEdge

Represent a future edge in the CFG

This type of edges is used when we don't know yet the source or destination block so only the address are stored. Every of the pending edges is expected to disappear after the whole program has been analyzed and each address translated to a block index or a call reference.
