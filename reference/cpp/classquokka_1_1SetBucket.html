template<std::derived_from< ProtoHelper > P>
class quokka::SetBucket< P >

quokka::SetBucket

Bucket representation

A bucket is a deduplicated container where every element is only stored once and everytime a new element already existing is added, the reference count is incremented.

Note
It ensures pointer stability
