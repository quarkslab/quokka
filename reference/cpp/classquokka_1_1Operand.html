Detailed Description

quokka::Operand

Operand representation

This mostly replicate every field found in op_t class. The operands are deduplicated so only one instance will be stored (e.g only one "eax").

Note
We could go further and analyze expression inside the operand to deduplicate all of them (e.g [eax+0x5] and [eax+0x7] could be refactored) but it's not done yet.
